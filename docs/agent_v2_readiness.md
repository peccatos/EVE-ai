# Agent v2 Readiness

Command:

```bash
cargo run -- agent-v2-readiness
```

Readiness requires CI proof, OpenAI adapter presence, rule-based fallback,
structured proposal validation, proposal preview, dry-run, code edit engine,
repo map, repo-aware planner, task outcome memory, validation, TUI visibility,
and safety policy.

No API key is required for basic operation.
